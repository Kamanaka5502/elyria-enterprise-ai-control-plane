# Visual Index

## Elyria Enterprise AI Control Plane

This index gives the repository a fast executive scan path.

---

## Control Plane Signal

| Signal | Meaning |
|---|---|
| **Enterprise AI** | The repository addresses organization-scale AI adoption, not isolated prompt experiments. |
| **Responsible AI** | AI movement is evaluated through ownership, authority, data, control, evaluation, observability, and revalidation. |
| **Azure Alignment** | The architecture maps to Azure AI, Microsoft Purview, Entra ID, Azure OpenAI, Azure AI Search, Databricks, CI/CD, and observability patterns. |
| **Pilot Ready** | The repository includes sandbox scenarios, readiness checklists, operating model artifacts, and executive demo material. |

---

## Color-Coded Decision Model

| Decision | Color Signal | Meaning |
|---|---|---|
| **ADMIT** | Green / teal | Movement may advance under current controls. |
| **HOLD** | Gold | Evidence or control coverage is incomplete. |
| **REFUSE** | Graphite / hard stop | Critical boundary is missing; movement cannot advance. |
| **REVALIDATE** | Steel blue | Conditions changed; prior approval must be reviewed. |
